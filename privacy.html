Politique de Confidentialité

1. Introduction

Bienvenue sur EZ JOB, une plateforme d'entraînement et de coaching en développement web. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données personnelles lorsque vous utilisez notre application.

2. Données collectées

EZ JOB fonctionne principalement de manière locale sur votre appareil. Les données suivantes peuvent être stockées dans le stockage local de votre navigateur :

  • Nom du joueur : Le nom que vous saisissez pour personnaliser votre expérience.
  • Progression et scores : Vos résultats aux quiz, historique des sessions, et statistiques de jeu.
  • Trophées et récompenses : Informations sur les trophées débloqués et vos points accumulés.
  • Préférences de jeu : Paramètres comme le temps par question ou le mode de jeu choisi.

Important : Aucune de ces données n'est transmise à nos serveurs ou à des tiers. Tout reste stocké localement sur votre appareil.

3. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Fournir une expérience de jeu personnalisée et fluide.
  • Sauvegarder votre progression entre les sessions.
  • Afficher vos statistiques et trophées.
  • Améliorer l'interface et les fonctionnalités de l'application.

4. Partage des données

EZ JOB ne partage, ne vend ni ne loue vos données personnelles à des tiers. Toutes les données restent strictement locales à votre appareil.

5. Sécurité des données

Bien que les données soient stockées localement, nous vous recommandons de :

  • Utiliser un navigateur à jour avec les dernières mises à jour de sécurité.
  • Éviter de partager votre appareil avec des personnes non autorisées.
  • Nettoyer régulièrement le cache de votre navigateur si nécessaire.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants concernant vos données :

  • Droit d'accès : Vous pouvez consulter les données stockées dans le stockage local de votre navigateur.
  • Droit de rectification : Modifiez votre nom ou vos préférences directement dans l'application.
  • Droit à l'effacement : Supprimez toutes vos données en vidant le cache de votre navigateur ou en utilisant les outils de développement.
  • Droit à la portabilité : Exportez vos données si vous le souhaitez (via les outils du navigateur).

Pour exercer ces droits ou poser des questions, contactez-nous à : contact@alternative-rvb.com

7. Cookies et technologies similaires

EZ JOB n'utilise pas de cookies traditionnels. Cependant, le stockage local du navigateur (localStorage) est utilisé pour sauvegarder vos données. Vous pouvez désactiver cette fonctionnalité dans les paramètres de votre navigateur, mais cela pourrait limiter l'expérience de jeu.

8. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Les modifications seront publiées sur cette page avec une date de mise à jour.

9. Contact

Pour toute question concernant cette politique ou vos données, contactez-nous :

  • Email : contact@alternative-rvb.com
  • Site web : alternative-rvb.com
